## Коротко обо мне

Я учусь на прикладной математике и информатике в НИУ ВШЭ и развиваюсь в аналитике данных, продуктовой аналитике и прикладном ML. Мне интересны проекты, где нужно не просто построить график или модель, а пройти весь аналитический цикл: понять задачу, подготовить данные, проверить гипотезы, оценить качество и сделать выводы, которые можно объяснить человеку.

Сейчас мой фокус:

- Python для анализа данных: `pandas`, `NumPy`, `scikit-learn`, визуализация;
- SQL и проектирование данных: PostgreSQL, CTE, оконные функции, ограничения, триггеры;
- BI и продуктовая аналитика: Yandex DataLens, метрики, дашборды, интерпретация;
- статистика и ML: проверка гипотез, ROC-AUC, feature importance, backtesting, риск-метрики.
